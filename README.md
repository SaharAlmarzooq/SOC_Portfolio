## 🔍 Investigation Methodology

Each investigation follows a structured SOC investigation workflow:

1. Review the investigation scenario.
2. Collect evidence.
3. Identify Indicators of Compromise (IOCs).
4. Analyze available artifacts and logs.
5. Correlate events across multiple data sources.
6. Assess the overall impact.
7. Classify the incident.
8. Determine whether escalation is required.
9. Document findings and recommendations.
